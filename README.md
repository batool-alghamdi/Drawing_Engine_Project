## Features 
- Draw, resize, and drag shapes.
- Change pen settings.
- Change shapes' style.
- Draw by mouse, and by source code.
- Save drawing canvas as source code with the extension (.drw), and open .drw files and render them in the canves.
- Clear the canvas.
